## Overview
This project is a JUCE-based audio plugin implementing a mathematically modeled preamp inspired by the TASCAM Portastudio 424 MKI.

## Features
- Real-time audio processing using JUCE
- Gain staging behavior modeled from analog preamp characteristics
- Custom signal processing pipeline

## Purpose
This project explores:
- Translating analog circuit behavior into digital signal processing
- Real-time audio system design
- Audio plugin architecture using C++

## Status
This is an early-stage implementation of a modeled preamp. Future work will focus on improving accuracy, nonlinear behavior, and full circuit simulation.
